# Infinite Canvas

_An infinite scrolling interface that recreates a 3D depth effect using a high-performance custom 2D Canvas engine. Based on the original 3D implementation by Edoardo Lunardi for Codrops._

![Image Title](https://tympanus.net/codrops/wp-content/uploads/2025/12/edoardo-lunardi-infinite-canvas-featured-image.jpg)

[Article on Codrops](https://tympanus.net/codrops/?p=106679)

[Demo](https://tympanus.net/Tutorials/InfiniteCanvas/)

## Features

- **Custom 2D Engine** - Native HTML5 Canvas implementation that recreates 3D depth
- **Infinite Space** - Navigate through an endless grid of media items
- **Performance Optimized** - Chunk-based rendering with distance-based culling
- **Touch & Mouse Controls** - Drag/pinch to navigate with inertia and drift
- **Keyboard Navigation** - WASD to move, QE to pan vertically

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- TypeScript
- Vite
- HTML5 Canvas
